# Информация о железках и windows


### Информация о Windows в командной строке

     C:\>systeminfo


<br/>

### Информация о BIOS в командной строке

    wmic bios get smbiosbiosversion

Можно еще вызвать:

    msinfo32

Далее: System Summary --> Bios Version


<br/>

### CPU-Z
