# Инсталляция Ansible в Ubuntu 14.04

Делаю: 26.10.2018

    $ sudo apt-add-repository ppa:ansible/ansible

    $ sudo apt-get update

    $ sudo apt-get install -y ansible

    $ ansible --version
    ansible 2.5.0
