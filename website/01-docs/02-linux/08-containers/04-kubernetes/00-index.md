# Kubernetes

Kubernetes converts a set of computers into one big one.

<br/>

**Building blocks:**

- RCs - Replication Controllers  
- PODS - минимальная единица управляемая RC.
- Services

<br/>

**Kubernets:**

- One instance for the Master
- Serveral instances as Minions
