# Программирование в Linux на GO


### Инсталляция GO в Ubuntu like дистрибутивах


1.install GO:

    $ sudo apt-get install -y golang

<br/>

 2.Set $GOPATH:

     $ export GOPATH=~/go  
     $ mkdir ~/go

<br/>

 3.Пример инсталляции на примере SWARM:

    $ go get -u github.com/docker/swarm

<br/>

 4.Update $PATH:

     $ export PATH=$PATH:~/go/bin

<br/>

 5.Virify install:

    $ swarm --version
