### Настройка сети в CoreOS

    $ sudo su -

    # vi /etc/systemd/network/static.network

<br/>

    [Match]
    Name=enp0s8

    [Network]
    Address=192.168.1.11/24
    Gateway=192.168.1.1
    DNS=192.168.1.1


<br/>

    # systemctl restart systemd-networkd

Не помогло, пришлось рестартовать.

<br/><br/>

    $ ssh core@192.168.1.11
