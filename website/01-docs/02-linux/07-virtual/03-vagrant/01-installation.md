# Инсталляция Vargant в Ubuntu 14.04


DOWNLOAD VAGRANT:  
https://www.vagrantup.com/downloads.html

<br/>

### Инсталляция Vagrant

    # cd /tmp
    # wget https://releases.hashicorp.com/vagrant/1.8.7/vagrant_1.8.7_x86_64.deb

    # dpkg -i vagrant_1.8.7_x86_64.deb

    $ vagrant -v
    Vagrant 1.8.7
